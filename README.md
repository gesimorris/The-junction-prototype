# junction - Kamloops Art Party Prototype

For some context of why this project is being developed: - [Arc Technologies Hackathon 2025](https://gesilogbook.hashnode.dev/arc-tech-hackathon-2025)

This is a platform that allows users to find local events they are interested in also seeing who is also attending to form a friend group at the event without ever talking. An event is posted on the platform, user sees the event and is interested in attending but doesn't have anyone to go with. They see other people just like them who are going to the event and form a group with some of these people based on their interests. Can message in a group chat before the event, finally meet at the event. User ratings as well as other metrics.
